Artificial Intelligence in Manufacturing - Examples

Manufacturers use AI to analyse sensor data and predict breakdowns and accidents. Synthetic intelligence systems aid production facilities in determining the likelihood of future failures in operational machinery, allowing for preventative maintenance and repairs to be scheduled in advance. AI also drives safety related projects,thus reducing the chances of accidents, Given below is details about using camera based solution in Industry

Object Detection using AI Architecture

Image read from the camera is processed using AI algorithms implemented in python language to detect the presence of human and gives a trigger

The Processor running the program was named as Flagman

These logics were implemented using a jetson nano controller,which provide a 5v logic output using the GPIO pins of the controller

5v output was used to energize a relay and this input was connected to our PLC to provide a safety interlock to the machine

Hand Safety in Industries using AI

Whenever human hand is detected in the area marked, AI System sends a command to stop the machine

This helps in reducing the risk associated with saw operation

Human Safety in Industries using AI

If AI based camera system detects human presence in Work station

Then it will trigger a command to close the boom barrier which will eventually avoid forklift entry to the station during human presence
